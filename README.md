 - [Strongloop/Loopback framework version 3](https://strongloop.com/)

# Run services
Make sure the MongoDB serivce is already running.
At the root directory of the repositoty
```
$ cd auth_server
$ npm install
$ slc start
$ cd ../api_server
$ npm install
$ slc start
```
